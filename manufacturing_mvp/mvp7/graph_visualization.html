物料流向

    🎯 场景说明

    流程描述:
    1. 原料储罐 → 过滤器 → 泵 → 预热器
    2. 预热器 → 反应釜（聚合反应）
    3. 反应釜 → 冷凝器 → 精馏塔
    4. 精馏塔 → 过滤器 → 成品储罐
    🔮 推理作用:
    通过物料流向传递性推理，自动发现设备间的连接关系和潜在瓶颈。例如：如果泵PMP-001进料至预热器HEX-001，预热器进料至反应釜RCT-001，则可推断泵间接向反应釜供应原料。

    ⚙️ 工艺约束

    温度约束: 反应釜温度 < 200°C
    压力约束: 反应釜压力 < 3.0MPa
    液位约束: 储罐液位 10%-90%
    泵约束: 出口压力 > 入口压力
